English

Cascading Style Sheets is a mechanism for adding style to a web document. The CSS code can be applied directly to the tags or be contained within the tags. It is also possible, instead of placing the formatting inside the document, to link to a CSS file that contains the styles.

Português

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web. O código CSS pode ser aplicado diretamente nas tags ou ficar contido dentro das tags. Também é possível, em vez de colocar a formatação dentro do documento, criar um link para um arquivo CSS que contém os estilos.
